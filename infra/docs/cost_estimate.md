> **前提**: 東京リージョン（ap-northeast-1）、2025年5月時点の料金。  
> トラフィック・ストレージはゼロ起点の最小構成で試算。実際の利用量に応じて変動する。  
> 為替レートは 1USD = 155円で換算。

## コスト削減のポイント

### dev 環境

- **ALB を使わない時間帯に destroy する**  
  ALB は起動しているだけで $18/月 かかる。開発中のみ apply、夜間・週末は destroy で節約できる。

- **RDS を使わない時間帯に停止する**  
  RDS は手動停止可能（最大7日で自動再起動）。停止中はストレージ料金のみ。

### prod 環境

- **NAT Gateway のコストに注意する**  
  NAT Gateway は $90/月 と最大のコスト要因。ECS を パブリックサブネットに移動すれば削減できるが、セキュリティのトレードオフがある。

- **Fargate Spot の活用**  
  タスク定義に Fargate Spot を混在させることで ECS コストを最大 70% 削減できる。

---

## 注記

- 上記はトラフィック・データ転送がほぼゼロの**最小構成**での試算。
- 実際のトラフィックが増えると CloudFront・ALB の LCU・データ転送料金が加算される。
- AWS 無料枠（12ヶ月）が適用される場合は一部サービスがさらに安くなる。
- 料金は AWS の改定により変動するため、最新値は [AWS 料金計算ツール](https://calculator.aws/pricing/2/home) で確認すること。
